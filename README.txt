## Implementation of ML Techniques in Python

#####To see output, simply run `TextParser.py`.

This is a Python 3.6.0 code.

#####Code structure:
1.  `TextParser.py` includes utility methods like reading files,
    extracting features, obtaining data, pre-processing
    text (for Amazon review), and natural language processing.
    It also has methods and library calls to different ML
    techniques
2.  `grapher.py` plots observed results
    1.  The input files are placed in `data` folder. A training 
    and a test data set are placed in it.
